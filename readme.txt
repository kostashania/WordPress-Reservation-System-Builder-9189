== Description ==

Table Reservation Builder is a powerful WordPress plugin that allows you to create and manage table reservations with ease. Perfect for restaurants, cafes, bars, and any business that needs to manage table bookings.

**Key Features:**

* **Easy Form Builder** - Create beautiful reservation forms with drag-and-drop simplicity
* **reCAPTCHA v3 Protection** - Advanced spam protection with invisible reCAPTCHA
* **Email Notifications** - Automatic notifications to admin and customers
* **Admin Dashboard** - Complete reservation management system
* **Responsive Design** - Forms work perfectly on all devices
* **Customizable Styling** - Multiple color schemes and form styles
* **Shortcode Integration** - Easy integration with any page or post
* **Database Storage** - All reservations stored securely in your WordPress database

**Form Fields Available:**
* Date picker with date validation
* Time picker with time constraints
* Guest count selector
* Customer name (required)
* Customer email (required)
* Customer phone (optional/required)
* Special requests textarea

**Admin Features:**
* View all reservations in organized dashboard
* Filter reservations by date, status, etc.
* Export reservation data
* Bulk actions (confirm, cancel, delete)
* Email template customization
* Complete settings management

**Security & Reliability:**
* reCAPTCHA v3 integration for spam protection
* Data validation and sanitization
* SQL injection protection
* XSS prevention
* Rate limiting support

**Easy Integration:**
Use simple shortcodes to add reservation forms anywhere:
* `[table_reservation_form]` - Display the reservation form
* `[table_reservation_list]` - Show recent reservations
* `[table_reservation_calendar]` - Display reservation calendar

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/table-reservation-builder/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to 'Reservations' > 'Settings' to configure the plugin
4. Add the shortcode `[table_reservation_form]` to any page or post

== Frequently Asked Questions ==

= How do I add a reservation form to my website? =

Simply add the shortcode `[table_reservation_form]` to any page, post, or widget where you want the form to appear.

= Can I customize the form appearance? =

Yes! The plugin includes multiple color schemes and styling options. You can also override the CSS for complete customization.

= Does this work with reCAPTCHA? =

Yes, the plugin includes full reCAPTCHA v3 integration for spam protection. You'll need to get your keys from Google reCAPTCHA.

= How are reservations stored? =

All reservations are stored in your WordPress database with full data security and backup compatibility.

= Can I export reservation data? =

Yes, you can export reservations from the admin dashboard in various formats.

= Is the plugin translation ready? =

Yes, the plugin is fully translation ready and includes .pot files for easy localization.
